# PMS Sitemap - current Next.js app

Ghi chú:
- `/` tự điều hướng: đã đăng nhập -> `/dashboard/overview`, chưa đăng nhập -> `/login`.
- `(auth)`, `(dashboard)`, `(admin)` là route group nội bộ của Next.js, không xuất hiện trên URL.
- Các route dưới đây là những màn hình/API còn tồn tại trong `src/app` hiện tại.
- Các chức năng filter, comment, approve, attachment, delete nằm trong màn hình tương ứng, không phải route riêng.
- Detail URL của tài liệu dùng segment rút gọn dạng `/projects/[ma-du-an]/modules/[ten-module]/documents/[ten-tai-lieu]`; route cũ chứa ID vẫn được nhận và tự điều hướng về URL rút gọn.

Main user flow:

[Login]
  -> [/dashboard/overview]
  -> [/projects]
  -> [/projects/:projectId/overview]
  -> [Sidebar project: Tổng quan dự án / BI Dashboard / Tài liệu / Export JSON]
  -> [/projects/:projectId/modules/:moduleId/documents]
  -> [/projects/:projectId/modules/:moduleId/documents/:docId]
  -> [Edit / History / Comment / Task liên quan]
